# PDF 页图 WebP 基准需求

- 日期：2026-03-22
- 主题：PDF 页图编码格式与 WebP 质量参数决策

## 目标

用本地运行数据判断：
1. PDF 页图是否应默认使用 `image/webp`
2. 如果保留 WebP，质量参数应设为多少

## 约束

- 不回滚工作区内无关修改
- 只评估 `canvas -> toBlob -> blob URL` 这一步
- 明确区分编码成本与 `pdf.js page.render()` 光栅化成本

## 验收标准

- 给出至少一组当前尺寸和一组 2.5x 目标尺寸数据
- 对比 JPEG / WebP / PNG
- 给出可执行的默认决策与质量参数建议

# PDF 页图热路径回退与动态缩放需求

- 日期：2026-03-22
- 主题：PDF 页图热路径回退 `JPEG@0.88`，并引入动态渲染缩放策略

## 目标

1. 将 PDF 页图热路径从 `WebP` 回退到 `JPEG@0.88`
2. 保留仅供开发者调试的隐藏编码覆盖标志，不向用户暴露开关
3. 将固定 `MAX_RENDER_SCALE` 改为基于页面尺寸和 DPR 的动态缩放策略
4. 用参数化回归测试覆盖至少三种 PDF 页面类型

## 约束

- 不回滚工作区中无关修改
- `WebP` 不得进入默认热路径
- 调试标志只允许开发者侧使用
- 验证必须包含类型检查与定向测试

## 验收标准

- 默认编码调用变为 `image/jpeg`，质量 `0.88`
- `JPEG` 编码失败时仍能回退到 `PNG`
- 动态缩放在不同页面类型下返回不同结果，而非固定常数
- 回归测试至少覆盖紧凑文本页、标准书页、宽屏幻灯片三类页面
